# DreamShot Multi-Lingual Bot
Zero-to-hero serverless Telegram bot that turns user photos into cinematic dreamy images in 5 seconds.

## Languages
English | فارسی | Русский | العربية | हिन्दी

## Deploy
1. Fork this repo
2. Connect to Render.com dashboard
3. Fill `BOT_TOKEN`, `HF_TOKEN`, `PAYMENT_LINK` in dashboard
4. Hit Deploy – webhook set automatically
